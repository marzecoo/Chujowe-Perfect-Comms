# Release Notes Template

Use this divider in every versioned changelog or release-notes page, directly after the opening summary and before the first section heading.

```html
<p align="center">
  <img src="https://raw.githubusercontent.com/marzecoo/Chujowe-Perfect-Comms/vX.Y.Z/assets/brand/divider.svg" alt="divider" width="900">
</p>
```

Replace `vX.Y.Z` with the release tag, for example `v2.0.2`.
