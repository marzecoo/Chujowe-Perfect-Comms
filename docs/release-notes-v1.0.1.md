## Fork Changes

- Built as `MegaChujowePerfectComms.dll`.
- Update popups and patch notes point to the Mega Chujowe Perfect Comms fork releases.
- Added TouMCE role voice support for Pelican belly voice, Infiltrator recruits, Lawyer and Client, Spirit Master, and Hacker Jam.
- Added Team Radio channels for Recruits and Lawyer/Client, alongside the existing Impostor, Vampire, and Lover channels.
- Added an option for Pelican victims to talk only inside the belly. When disabled, swallowed players are muted.
- Added Spirit Master ghost voice modes like Medium.
- Added one hidden-role mute option covering Swooper-style hidden roles, including Astral, Burrower, Speedy, Vanisher, and Wraith.
- Added Hacker Jam voice muting.
- Increased speaker and per-player volume controls up to 300%.
- Ported upstream Perfect Comms fixes from 2.0.7 and 2.0.8, including exile meeting routing, safer unavailable-player routing, camera hearing fixes, and stronger volume/mute handling.

## Notes

Windows Defender or another antivirus may flag mod DLLs because this build embeds voice/network libraries inside one plugin DLL. Download only from this fork release page and do not mix it with the original Perfect Comms DLL.
